# Assets — GDAC Platform

Arquivos de mídia utilizados na landing page da GDAC e nas landing pages dos parceiros.

## Estrutura

```
Assets/
  images/    → fotos com licença Adobe (JPG, PNG, WebP, SVG)
  videos/    → vídeos com licença Adobe (MP4, WebM, MOV)
  logo/      → logotipos da GDAC (PNG, SVG, variações claro/escuro)
```

## Licenciamento

As imagens e vídeos nas pastas `images/` e `videos/` possuem **licença de uso Adobe**.  
Não redistribuir, não publicar em repositórios públicos, não usar fora do escopo da plataforma GDAC.

## Observações

- Formatos recomendados para web: **WebP** (imagens) e **MP4/H.264** (vídeos)
- Resolução mínima para banners: **1920×1080px**
- Vídeos de fundo (hero): sem áudio, loop, máximo **10MB** após compressão
